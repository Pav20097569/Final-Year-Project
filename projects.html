Real-Time OBD-II Data Visualization and Analysis Platform for Vehicle Diagnostics
Commercial Title
DriveSense: Smarter Car Care Through Real-Time Data
Project Description

This project develops a real-time OBD-II data monitoring system to provide instant insights into a vehicle's health and performance. Using Vector CANoe to simulate data, the system collects and processes live sensor data (e.g., RPM, coolant temperature, DTCs) and displays it on a user-friendly interface. The system also supports real-time alerts for critical issues, ensuring drivers can address problems immediately.

The system integrates cloud storage and AI-based algorithms for predictive maintenance and actionable insights. Built on a Raspberry Pi and Python, it offers a cost-effective, scalable solution for modern vehicle diagnostics, empowering drivers to make informed decisions and maintain their vehicles effectively.
